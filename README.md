# Linux Host-Based Firewall (Mini Project)

This project implements a host-based firewall on Linux using Netfilter and iptables.
It follows a default deny security model and allows only required traffic.

## Objectives
- Implement firewall rules on Linux
- Control incoming and outgoing traffic
- Log dropped packets
- Ensure persistence using systemd
- Automate rule management

## Tools & Technologies
- Linux (Kali/Ubuntu)
- Netfilter
- iptables
- Bash scripting
- Python
- systemd
- GitHub
